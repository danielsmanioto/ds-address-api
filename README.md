# ds-address-api
API for centralize Address rules and CEP search 

### Compilar e subir a aplicação
`mvn clean install && mvn spring-boot:run`

### Buscar endereco por CEP
`http://localhost:8080/address/cep/<CEP>`

### APIs utilizadas
https://viacep.com.br/

## Tecnologias
Java 8+ <br>
SpringBoot 2<br>
Maven<br>
